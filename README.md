# Loan-Approval

Loan Approval Prediction App – Project Note

This project is a Machine Learning based Loan Approval Prediction System which predicts whether a loan will be Approved or Rejected based on applicant details.
The web application is built using Streamlit, and the ML model is trained using Python and Scikit-Learn, then saved as model.pkl.

Input Features Used

No. of Dependents

Education (Graduate / Not Graduate)

Self Employed (Yes / No)

Annual Income

Loan Amount

Loan Duration

CIBIL Score

Assets

Output

✔ Loan Approved (if model predicts 1)

❌ Loan Rejected (if model predicts 0)

Technologies Used

Python

Streamlit

Pandas

Scikit-Learn

Pickle (for saving model)
